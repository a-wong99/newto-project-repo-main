In this project, I acted as a Junior Data Analyst for a bookstore group which sells books across several US regions, and my job was to create a Q1 2024 performance review to understand which genres, regions and publishers are performing well, and to identify where to focus marketing and purchasing for Q2 2024.

The link below contains the link to the Tableau Public workbook, which contains all the visuals and dashboards I created for this project.
